# Network Security

**Grade:** VG

## Course Overview

This course focused on professional network security, covering both theoretical concepts and practical implementations through labs and exercises.

## What I Learned

- **Core Concepts** – CIA triad, access control and SIEM.
- **Security Terminology** – Key terms and concepts in network security.
- **Attack Techniques** – Man-in-the-middle attacks and privilege escalation.
- **Network Security** – Control plane security, NAT and NTP.
- **VPN** – Virtual Private Networks and secure communication.
- **Firewalls** – Configuration and management of firewalls.
- **IPS & IDS** – Intrusion Prevention and Detection Systems.

## Key Topics

- CIA triad and access control.
- SIEM implementation.
- Man-in-the-middle attack prevention.
- Privilege levels and access management.
- Firewall configuration.
- IPS and IDS setup and management.
- VPN configuration and security.
