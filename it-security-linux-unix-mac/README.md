# IT Security Linux, Unix, Mac

**Grade:** VG

## Course Overview

This course focused on Linux, Unix and macOS system technology and professional network security. The course covered administration, configuration and hardening of these systems to minimize possible attack surfaces with respect to installed services.

## What I Learned

- **Desktop & Server Security** – Security fundamentals for Linux, Unix and macOS.
- **Hardware & Network Security** – Securing hardware and network configurations.
- **Hacker Mindset** – Understanding how attackers think and operate.
- **System Hardening** – Hardening operating systems and identifying insecure applications and configurations.
- **Advanced Hardening** – Advanced hardening techniques and system monitoring.
- **Ethical Hacking** – Introduction to ethical hacking in Linux environments.
- **Preventive Security** – Proactive security work in Linux, Unix and macOS environments.

## Key Topics

- Administration and configuration of Linux, Unix and macOS.
- Minimizing attack surfaces.
- Identifying and securing insecure applications and configurations.
- Network security in Linux environments.
- Ethical hacking fundamentals.
- Advanced system monitoring.
