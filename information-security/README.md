# Information Security

**Grade:** G

## Course Overview

This course focused on laws, regulations and methods within information security and IT law. The course provided tools and methods to implement security routines and policies in the role of an IT Security Specialist.

## What I Learned

- **IT Law & Ethics** – Laws, regulations and ethics affecting information security.
- **ISO 27000** – The ISO 27000 series and its impact on information security work.
- **GDPR** – How GDPR and the Data Protection Regulation govern information security.
- **Standards** – IEC-62443 (OT security) and NIST 800-53.
- **Risk Analysis** – Identifying and assessing threats and risks in information security.
- **Security Policies** – Establishing information security policies and incident reporting plans.
- **Compliance** – Analyzing IT infrastructure compliance with applicable laws and regulations.

## Key Topics

- Information security laws and regulations.
- ISO 27000, GDPR, IEC-62443 and NIST 800-53.
- Risk analysis and threat assessment.
- Security policy development.
- Incident reporting plans.
- Internet and web security regulations.
