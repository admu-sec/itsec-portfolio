# Advanced IT Security

(2025-10-27 → 2025-11-28)

**Grade:** VG

## Course Overview

This course covered advanced IT security concepts including cloud security, offensive security tools, advanced file permissions, sandboxing and Windows security. The course combined theoretical knowledge with practical lab work and agile project methodology.

## What I Learned

- **Offensive Security** – Metasploit payloads, handlers and exploitation flows, Nmap port scanning.
- **Cloud Security** – Microsoft Azure security services including VNET, Peering, NSG and ASG.
- **Advanced File Permissions** – SUID, SGID, sticky bit and GUID in Linux environments.
- **Sandboxing** – Application isolation techniques and security implications.
- **Application Security** – OWASP Top 10 vulnerabilities and mitigations.
- **Windows Security** – Group Policy (GPO), Windows Defender Credential Guard and CVE exploitation.
- **Agile Security** – Agile methodologies applied to cybersecurity and security by design.
- **AI in Security** – AI and LLM tools in SOC environments and associated risks.

## Key Topics

- Metasploit and Nmap in offensive security.
- Azure cloud security architecture.
- Advanced Linux file permissions and sandboxing.
- Demonstrating and analyzing Windows CVEs.
- Group Policy and Windows Server security.
- Security by design in agile development.
- AI tools in SOC workflows.
