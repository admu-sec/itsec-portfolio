# Cloud Security

**Grade:** VG

## Course Overview

This course focused on cloud security architecture and concepts, covering Microsoft Azure security services, network security, encryption, virtualization and access management in cloud environments.

## What I Learned

- **Cloud Computing** – IaaS, PaaS, SaaS, public, private and hybrid cloud models.
- **Network Security** – NAT, ACL, NSG, routing protocols and Azure networking.
- **Firewalls** – Azure Firewall and Fortigate firewall configuration and comparison.
- **Redundancy** – STP, FHRP, disaster recovery and high availability solutions.
- **Encryption** – Symmetric and asymmetric encryption, hashing, PKI, root CA and certificates.
- **VPN** – Remote VPN and Site-to-Site VPN in hybrid environments.
- **RBAC** – Role Based Access Control in Azure, VMware and Hyper-V.
- **IPv6** – IPv6 networking and security implications in cloud environments.
- **Virtualization** – VMware, Hyper-V and nested virtualization environments.
- **Logging & Monitoring** – Log management, vCenter monitoring and AI-assisted security.
- **Azure Key Vault** – Key and secret management in Azure.
- **XDR** – Extended Detection and Response with AI capabilities.

## Key Topics

- Azure cloud security architecture.
- Network security groups and access control lists.
- Firewall configuration and traffic management.
- Site-to-Site and Remote VPN setup.
- Encryption and certificate management.
- Role Based Access Control.
- Backup 3-2-1 rule and disaster recovery.
- Log analysis and monitoring in cloud environments.
