# Ethical Hacking

**Grade:** VG

## Course Overview

This course focused on web infrastructure security, covering how to identify, exploit and mitigate security vulnerabilities in web environments. The course began with understanding web infrastructure before moving on to ethical hacking techniques and incident management.

## What I Learned

- **Web Infrastructure** – How web servers communicate with browsers and databases.
- **OWASP** – Common web security threats and the OWASP Top 10.
- **Web Hacking** – Practical hacking exercises targeting web servers.
- **Man-in-the-Middle Attacks** – Understanding and preventing MITM attacks.
- **Monkeypatching** – Runtime modification of code and security implications.
- **Logging & Incident Reporting** – Monitoring and responding to security incidents.
- **Defensive Techniques** – Securing and hardening web environments against attacks.

## Key Topics

- Web server architecture and communication.
- OWASP security threat categories.
- Practical web hacking exercises.
- Man-in-the-middle attack techniques and prevention.
- Log analysis and incident reporting.
- Identifying and mitigating web vulnerabilities.
