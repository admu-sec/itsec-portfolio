# Network Technology

(2024-08-26 → 2024-09-20)

**Grade:** G

## Course Overview

This course focused on network technology, covering configuration of routers and switches for improved network security, building networks from scratch, and identifying and addressing vulnerabilities in both Linux and Windows environments.

## What I Learned

- **Network Fundamentals** – OSI model, TCP/IP, LAN technologies.
- **Switching** – VLAN configuration, STP (Spanning Tree Protocol).
- **Routing** – Routing protocols, IP networking and inter-VLAN routing.
- **Wireless Security** – Access points, wireless network security and access control.
- **Network Architecture** – Planning, configuring and troubleshooting networks.
- **Tools** – Cisco Packet Tracer for network simulation.

## Key Topics

- Router and switch configuration.
- VLAN and access lists.
- Wireless network security.
- Network planning and deployment.
- Fortinet Networking Fundamentals.
