# IT Infrastructure

**Grade:** VG

## Course Overview

This course focused on creating and building IT infrastructures with high availability, proper capacity and strong security. The course also covered efficiency optimization in IT systems and how an IT infrastructure platform is built with servers and backup solutions.

## What I Learned

- **System Administration** – High availability, capacity planning and IT security principles
- **Servers & Backup** – Server architecture, backup solutions and data management
- **Virtualization** – PC virtualization, server virtualization, data virtualization and application virtualization
- **IT Security** – Identifying vulnerabilities in IT infrastructure
- **IT History** – IBM systems, 7044/44M, System P and legacy infrastructure
- **Scalability** – Improving efficiency, cost savings and scalability in IT systems

## Key Topics

- Building IT infrastructures with high availability
- Capacity planning and resource allocation
- Server and backup platforms
- Multiple virtualization techniques
- Identifying and distributing computer resources across simultaneous applications
- Finding vulnerabilities in IT infrastructure
