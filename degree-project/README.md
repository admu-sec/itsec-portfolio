## Research Question

How can an open source-based SIEM solution be configured and calibrated to optimize detection capability?


## Overview

This degree project focuses on implementing and configuring Wazuh as an open source SIEM solution. The project involves creating custom detection rules, tuning the system to reduce false positives and mapping detections to the MITRE ATT&CK framework.
