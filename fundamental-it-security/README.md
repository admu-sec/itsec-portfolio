# Fundamental IT Security

(2024-11-04 → 2024-12-13)

**Grade:** G

## Course Overview

This course focused on protecting IT environments, systems and data. The course covered analysis and management of security risks related to the internet and cloud services.

## What I Learned

- **Authentication** – Single Sign-On (SSO), two-factor and three-factor authentication.
- **Malicious Code** – Understanding and handling malware and malicious software.
- **Firewalls** – Configuring different types of firewalls.
- **Cryptography** – Encryption, signing, PKI and digital certificates.
- **Cloud Security** – Security risks related to cloud services.
- **Vulnerability Detection** – Identifying and addressing vulnerable communication with remote hosts.

## Key Topics

- User authentication and access control.
- Firewall configuration.
- Creating and managing digital certificates.
- Encrypting and signing critical data.
- Security risk analysis and management.
