# IT Security Windows

(2025-01-20 → 2025-02-14)

**Grade:** VG

## Course Overview

This course focused on Windows system technology and configuration, covering how to minimize attack surfaces through hardening, securing system services and reducing the risk of cyberattacks.

## What I Learned

- **Windows Configuration** – System technology and implementation of Windows environments.
- **System Hardening** – Minimizing attack surfaces and securing Windows systems.
- **Security Footprint** – Update routines and password policies.
- **Problem Analysis** – Analyzing and improving systems during active operation.
- **Device Management** – Managing devices in a Windows environment.
- **LDAP & Active Directory** – Identity management and directory services.
- **Logging & Auditing** – Monitoring and auditing Windows systems.

## Key Topics

- Windows system configuration and implementation.
- Hardening and attack surface minimization.
- Password policies and update routines.
- LDAP, identity and access management.
- Logging and auditing.
- Preventing and mitigating cyberattacks.
